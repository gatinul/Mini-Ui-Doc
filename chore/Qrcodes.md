# Qrcodes 二维码生成
用于生成指定内容及长宽的二维码

## 如何使用

```html
<Qrcodes
  content="http://broad.10010lt.com/chbb/e10010/online/index.do?ID=20180309130005082820"
  width="150"
  height="150"
  >
</Qrcodes>
```

## API

|属性|说明|类型|默认值|
|--|--|--|--|
|content|二维码内容|String||
|width|二维码图片宽度，不带单位|String|150|     
|height|二维码图片高度，不带单位|String|150|
