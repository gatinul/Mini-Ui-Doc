Dot 徽章

Dot 徽标

单行右侧的红点、数字或者文字。用于告知用户，该区域的状态变化或者待处理任务的数量。

如何使用

  <Dot :text="dot"></Dot>

API

属性 说明 类型 默认值
text 要显示的文字或数字，为point时，显示红点 String
